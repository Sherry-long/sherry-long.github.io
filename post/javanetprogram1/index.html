UDP网络编程

UDP协议 数据传输
public class UDPTest {

    //发送端
    @Test
    public void send() throws IOException {
        //TCP使用Socket，UDP使用DatagramSocket
        DatagramSocket ds = new DatagramSocket();
        byte[] data = "this is a sentence".getBytes();
        //将要传输的数据封装在DatagramPacket中
        DatagramPacket packet = new DatagramPacket(data, 0, data.length, InetAddress.getLocalHost(), 8989);
        //使用DatagramSocket对象传输packet
        ds.send(packet);

        ds.close();
    }

    //接收端
    @Test
    public void receive() throws IOException {
        //指定DatagramSocket端口
        DatagramSocket ds = new DatagramSocket(8989);
        byte[] data = new byte[100];
        //使用DatagramPacket对象接收数据
        DatagramPacket packet = new DatagramPacket(data, 0, data.length);
        ds.receive(packet);

        System.out.println(new String(packet.getData(), 0, packet.getLength()));

        ds.close();
    }
}

URL编程

URL(Uniform Resource Locator)：统一资源定位符，它表示Internet上某一资源的地址

它是一种具体的URI，即URL可以用来标识一个资源，而且还指明了如何locate这个资源

通过URL我们可以访问Internet上的各种网络资源，浏览器通过解析URL可以在网络上查找相应的文件或其他资源

  • URL的基本结构由5部分组成
<传输协议>://<主机名>:<端口号>/<文件名>#片段名?参数列表
	>例如：
		http://192.168.1.100:8080/helloworld/index.jsp#a?username=shkstart&password=123
	>#片段名：即锚点
	>参数列表格式：参数名=参数值&参数名=参数值....
URL类常用方法
@Test
    public void URLtest(){
        URL url = null;
        try {
            url = new URL("http://192.168.1.100:8080/helloworld/index.jsp#a?username=shkstart&password=123");
            //获取协议
            System.out.println(url.getProtocol());
            //获取主机名
            System.out.println(url.getHost());
            //获取文件路径
            System.out.println(url.getPath());
            //获取端口号
            System.out.println(url.getPort());
            //获取文件名
            System.out.println(url.getFile());
            //获取查询名
            System.out.println(url.getQuery());
        } catch (MalformedURLException e) {
            e.printStackTrace();
        }
        /**
         * http
         * 192.168.1.100
         * /helloworld/index.jsp
         * 8080
         * /helloworld/index.jsp
         * null
         */
    }
URL网络编程实现Tomcat服务器文件下载
@Test
    public void URLDownload() {
        HttpURLConnection connection = null;
        BufferedOutputStream bos = null;
        try {
            //创建url对象
            URL url = new URL("http://localhost:8080/examples/huoyuanjia(instrumental).mp3");
            //获取http连接
            connection = (HttpURLConnection)url.openConnection();
            //连接
            connection.connect();
            //获取输入流，下载文件
            InputStream is = connection.getInputStream();
            bos = new BufferedOutputStream(new FileOutputStream("download.mp3"));

            byte[] data = new byte[1024];
            int len;
            while ((len = is.read(data)) != -1){
                bos.write(data, 0, len);
            }

            System.out.println("Download complete....");
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            if (bos != null) {
                try {
                    bos.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
            if (connection != null) {
                connection.disconnect();
            }
        }
    }
